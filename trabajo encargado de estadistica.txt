II. MARCO TEORICO
2.1  El fósforo como nutriente esencial
El fósforo (P) es uno de los macronutrientes primarios esenciales para el desarrollo de las plantas. Participa en procesos fundamentales como la fotosíntesis, la formación y desarrollo de raíces, la síntesis de ácidos nucleicos (ADN y ARN), la transferencia de energía (ATP y ADP) y la maduración de frutos y semillas.
La deficiencia de fósforo provoca un crecimiento lento, tallos delgados, raíces poco desarrolladas y hojas con tonalidades verde oscuro o púrpuras. En cambio, su adecuada aplicación promueve un crecimiento vigoroso y un mayor desarrollo del tallo y las raíces.
2.2. Importancia del fósforo en la agricultura
El estudio del fósforo en la carrera de Suelos y Agua es de gran relevancia, ya que permite comprender la dinámica de los nutrientes en el suelo y su relación con la productividad agrícola. El fósforo es un elemento poco móvil en el suelo, por lo que su disponibilidad depende de factores como el pH, el contenido de materia orgánica y la presencia de microorganismos solubilizadores de fósforo.
Un manejo adecuado de este nutriente contribuye a la fertilización sostenible, mejorando la eficiencia del uso de fertilizantes y evitando la contaminación de los cuerpos de agua por exceso de nutrientes.
2.3. Aplicación del análisis estadístico en experimentos agrícolas
En los estudios agronómicos, el análisis estadístico permite interpretar los resultados experimentales y tomar decisiones basadas en datos. La medición del largo del tallo de las plantas tratadas con fósforo constituye una variable cuantitativa continua, que puede analizarse mediante métodos estadísticos descriptivos.
Estos métodos ayudan a identificar la variabilidad del crecimiento vegetal y a comparar la respuesta de las plantas frente al tratamiento aplicado.
2.4. Tabla de distribución de frecuencias
La tabla de distribución de frecuencias es una herramienta estadística que organiza los datos obtenidos en intervalos, mostrando cuántas veces se repite cada rango de valores.
A través de esta tabla se pueden calcular diferentes parámetros como:
* Frecuencia absoluta (fi): número de veces que aparece un valor o intervalo.
* Frecuencia relativa (hi): proporción que representa cada valor respecto al total.
* Frecuencia acumulada (Fi y Hi): suma progresiva de las frecuencias absolutas o relativas.
Esta organización facilita el análisis y la comprensión de los resultados del experimento.
2.5. Representación gráfica de los datos
Los datos ordenados en una tabla de frecuencias pueden representarse mediante gráficos como:
* Histograma: muestra la distribución de los datos en intervalos.
* Polígono de frecuencia: une los puntos medios de las columnas del histograma.
* Ojiva: representa la frecuencia acumulada.
Estas representaciones gráficas permiten observar el comportamiento general del crecimiento de los tallos y detectar patrones o tendencias relacionadas con el efecto del fósforo.
2.6. Importancia del análisis estadístico en la interpretación de resultados
El análisis estadístico de las variables biológicas es fundamental para evaluar la eficacia de un tratamiento. En este caso, estudiar la distribución del largo del tallo de las plantas permite determinar la influencia del fósforo sobre su desarrollo.
Además, el uso de medidas de tendencia central (media, mediana, moda) y de dispersión (rango, varianza, desviación estándar) proporciona una visión más precisa del comportamiento de los datos, permitiendo interpretar si el crecimiento fue homogéneo o variable entre las plantas tratadas.

cat("\n============================================================\n")

# Para variables continuas, crear intervalos
n <- length(df$Largo_del_tallo_cm)

# Calcular número de intervalos (Regla de Sturges)
k <- ceiling(1 + 3.322 * log10(n))

cat("Número de datos (n):", n, "\n")
cat("Número de intervalos (k):", k, "\n\n")

# Crear intervalos
min_val <- min(df$Largo_del_tallo_cm)
max_val <- max(df$Largo_del_tallo_cm)
amplitud <- (max_val - min_val) / k

# Crear límites de intervalos
limites <- seq(min_val, max_val, length.out = k + 1)

# Crear intervalos con etiquetas personalizadas
intervalos <- cut(df$Largo_del_tallo_cm, 
                  breaks = limites, 
                  include.lowest = TRUE,
                  right = FALSE)

# Crear tabla de frecuencias completa
tabla_freq <- table(intervalos)
freq_abs <- as.vector(tabla_freq)
freq_rel <- prop.table(tabla_freq)
freq_porc <- prop.table(tabla_freq) * 100
freq_acum <- cumsum(freq_abs)
freq_rel_acum <- cumsum(freq_rel)
freq_porc_acum <- cumsum(freq_porc)

# Crear data frame con la tabla completa
tabla_completa <- data.frame(
  "Intervalo" = names(tabla_freq),
  "Límite Inferior" = round(limites[1:k], 2),
  "Límite Superior" = round(limites[2:(k+1)], 2),
  "Marca de Clase" = round((limites[1:k] + limites[2:(k+1)]) / 2, 2),
  "Frecuencia Absoluta (fi)" = freq_abs,
  "Frecuencia Relativa (hi)" = round(as.vector(freq_rel), 4),
  "Frecuencia Porcentual (%)" = round(as.vector(freq_porc), 2),
  "Frecuencia Acumulada (Fi)" = freq_acum,
  "Frecuencia Relativa Acumulada (Hi)" = round(freq_rel_acum, 4),
  "Frecuencia Porcentual Acumulada (%)" = round(freq_porc_acum, 2)
)

# Agregar fila de totales
fila_total <- data.frame(
  "Intervalo" = "TOTAL",
  "Límite.Inferior" = NA,
  "Límite.Superior" = NA,
  "Marca.de.Clase" = NA,
  "Frecuencia.Absoluta..fi." = sum(freq_abs),
  "Frecuencia.Relativa..hi." = round(sum(freq_rel), 4),
  "Frecuencia.Porcentual...." = round(sum(freq_porc), 2),
  "Frecuencia.Acumulada..Fi." = NA,
  "Frecuencia.Relativa.Acumulada..Hi." = NA,
  "Frecuencia.Porcentual.Acumulada...." = NA
)

tabla_completa <- rbind(tabla_completa, fila_total)
